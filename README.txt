==========================================
TROUBLESHOOTING
==========================================

Q: Invitation appears blank
A: Make sure all files are in the same folder

Q: Download not working
A: Use Print function and save as PDF instead

Q: Logo not showing
A: Verify dps-logo.png is in the same folder

Q: Form doesn't submit
A: Ensure all required fields are filled

Q: Title not added automatically
A: Check if you selected a position from dropdown
